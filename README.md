# ancient_poetry
Chinese ancient poetry database
##数据结构
| id          | 诗歌id            |
| title       | 诗歌标题          |
| emp         | 作者朝代          |
| author      | 作者              |
| text        | 诗歌正文          |
| titlepinyin | 标题（拼音全拼）  |
| textpinyin  | 正文（拼音全拼）  |
| titlepy     | 标题（首字拼音）  |
| textpy      | 正文（首字拼音）  |
